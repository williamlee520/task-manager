# 📋 任务管理器 - 云部署指南（Render.com 免费部署）

## 整体流程

```
代码上传 GitHub → Render 自动部署 → 获得永久公网地址 → 任何设备都能访问
```

---

## 第一步：上传代码到 GitHub

### 1.1 打开 GitHub 创建新仓库
- 访问 https://github.com/new
- 仓库名填 `task-manager`
- **不要**勾选 "Add a README"
- 点 **Create repository**

### 1.2 上传文件
创建完成后，页面上会有上传提示。最简单的方式：

1. 在仓库页面点 **"uploading an existing file"**
2. 把 `任务管理器.zip` 解压出来的所有文件拖进去：
   - `index.html`
   - `server.py`
   - `requirements.txt`
   - `render.yaml`
   - `.gitignore`
   - `manifest.json`
   - `sw.js`
   - （`启动.bat` 和 `启动.sh` 可选，不影响云部署）
3. 点 **Commit changes**

---

## 第二步：在 Render 部署（免费，不用绑卡）

### 2.1 注册 Render
- 访问 https://render.com
- 点 **Get Started** 用 GitHub 账号登录（推荐，这样能自动看到你的仓库）

### 2.2 创建服务
1. 登录后点 **New +** → 选 **Web Service**
2. 找到 `task-manager` 仓库，点 **Connect**
3. 配置页面：
   - **Name**: `task-manager`（随意，会变成网址的一部分）
   - **Runtime**: 自动识别为 Python
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `uvicorn server:app --host 0.0.0.0 --port $PORT`
   - **Plan**: 选 **Free**
4. 点 **Create Web Service**

### 2.3 等待部署
- 大约 2-3 分钟，状态从 `Building` → `Live`
- 部署成功后页面顶部会显示你的公网地址，类似：
  ```
  https://task-manager-xxxx.onrender.com
  ```

🎉 **这个地址就是你的任务管理器，任何设备任何网络都能访问！**

---

## 第三步：手机使用

1. 手机浏览器打开上面的地址
2. 点浏览器菜单 → **「添加到主屏幕」**
3. 桌面上就多了一个 App 图标，点开直接用

## 常见问题

### Q: Render 免费版有什么限制？
- 服务 15 分钟无访问会自动休眠，再次访问时冷启动约 **30 秒**
- 每月 750 小时免费时长（足够 24/7 运行一整个月）
- 数据存在 SQLite，**重启后数据不会丢失**（存在持久化磁盘上）

### Q: 冷启动 30 秒太慢怎么办？
两个办法：
1. 用免费的 [UptimeRobot](https://uptimerobot.com/) 每 5 分钟 ping 一次，保持服务不休眠
2. 升级 Render Starter 计划（$7/月），无休眠

### Q: 数据安全吗？
- SQLite 数据库存在 Render 服务器上，重启不丢失
- 但如果需要更高可靠性，建议定期用页面上的「导出」功能备份数据
- **重要数据请务必定期备份**

### Q: 可以自定义域名吗？
可以！在 Render 的 Settings → Custom Domains 里绑定你自己的域名。

### Q: 想更新代码怎么办？
1. 修改 GitHub 仓库里的文件
2. Render 会自动检测到更新并重新部署（约 2 分钟）
