# 📱 手机部署指南 - 只用手机完成

## 整体思路
把代码放到 GitHub → 让 Render 从 GitHub 拉代码自动部署 → 获得永久地址

---

## 第一步：注册账号（如果已有可跳过）

1. **GitHub** → 手机浏览器打开 https://github.com/signup 注册
2. **Render** → 手机浏览器打开 https://render.com 用 GitHub 登录

---

## 第二步：在 GitHub 创建仓库并上传文件

### 2.1 创建仓库
1. 打开 https://github.com/new
2. Repository name 填 `task-manager`
3. 其他都不勾，点 **Create repository**

### 2.2 上传文件
创建后页面会显示一个空仓库，点 **"creating a new file"** 或 **"upload files"**

**需要上传 3 个核心文件**，逐个创建：

#### 文件 1: `index.html`
- 点 **"creating a new file"**
- 文件名输入 `index.html`
- 内容：把解压 zip 得到的 index.html 的内容完整复制粘贴进去
- 点 **Commit changes**

#### 文件 2: `server.py`
- 同样方式，点 **"creating a new file"**
- 文件名输入 `server.py`
- 内容：复制粘贴 server.py 的内容
- 点 **Commit changes**

#### 文件 3: `requirements.txt`
- 点 **"creating a new file"**
- 文件名输入 `requirements.txt`
- 内容就 3 行：
```
uvicorn
fastapi
pydantic
```
- 点 **Commit changes**

---

## 第三步：Render 部署

1. 手机浏览器打开 https://dashboard.render.com
2. 点 **New +** → **Web Service**
3. 找到 `task-manager` 仓库，点 **Connect**
4. 填写配置：
   - **Name**: `task-manager`
   - **Runtime**: Python（自动识别）
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `uvicorn server:app --host 0.0.0.0 --port $PORT`
   - **Instance Type**: Free
5. 拉到最下面，点 **Create Web Service**
6. 等待 2-3 分钟，状态变成 **Live** 🎉

部署成功后，页面顶部会显示你的永久地址：
```
https://task-manager-xxxx.onrender.com
```

**这个地址任何设备、任何网络都能用！**

---

## 第四步：添加到手机主屏幕

1. 手机浏览器打开上面的地址
2. Safari：点底部分享按钮 → 添加到主屏幕
3. Chrome：点右上角菜单 → 添加到主屏幕
4. 桌面出现 App 图标，点开即用

---

## ⚡ 防止休眠（可选但推荐）

Render 免费版 15 分钟没人访问会休眠，再次打开要等 30 秒。

解决办法：
1. 打开 https://uptimerobot.com 注册
2. 创建一个 HTTP(s) 监控，填你的 Render 地址
3. 间隔设 5 分钟
4. 这样服务永远不会休眠，打开秒加载
